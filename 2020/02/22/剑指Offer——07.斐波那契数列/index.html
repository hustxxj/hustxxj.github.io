LeetCode练习（递归）：779.第K个语法字符

题目描述

在第一行我们写上一个 0。接下来的每一行，将前一行中的0替换为01，1替换为10。给定行数 N 和序数 K，返回第 N 行中第 K个字符。（K从1开始）

实现代码

第N排第K个，如果K<=2^(N-2),则等于N-1排第K个
如果k>2^(N-2),k-2^(N-2),等于N-1排第k-2^(N-2)个取反
